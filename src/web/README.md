# Monopoly.Web
Projekt gry Monopoly w wersji webowej.

## Instalacja

```sh
npm install
```

### Uruchomienie projektu

```sh
npm run dev
```

### Uruchomienie testów jednostkowych

```sh
npm run test:unit
```

### Uruchomienie testów end-to-end

```sh
npm run test:e2e:dev
```

### Uruchomienie linta

```sh
npm run lint
```

### Budowanie projektu

```sh
npm run build
```
